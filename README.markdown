# ActsAsViewable

### General information

Adds functionality to models.

## Installation

To use it, add it to your Gemfile:

    gem 'acts_as_viewable', :git => 'git@github.com:ivoreis/acts_as_viewable.git'
    bundle update

### Post Installation

1. rails g acts_as_viewable:install
2. rake db:migrate
